## Navegar

Primeiramente, navegue até o diretório fonte.

```bash
cd caminho/dos/arquivos
```

# Inicia o RMI Registry em uma nova janela/processo
```bash
start rmiregistry
```

# Inicia o Servidor
```bash
java MainServidor
```
